# Anti-Money-Laundering-system-using-Artificial-Neural-Networks-ANN-
This project focuses on detecting suspicious financial transactions and identifying potential money laundering activities using Artificial Neural Networks (ANN). The model is trained on transaction data to learn hidden patterns associated with fraudulent behavior and classify transactions as legitimate or suspicious. By leveraging machine learning techniques, the system enhances the efficiency of anti-money laundering (AML) processes, reduces manual investigation efforts, and supports financial institutions in regulatory compliance.

->Features
* Data preprocessing and feature engineering for transaction datasets.
* Artificial Neural Network (ANN) model for AML detection
* Classification of suspicious and non-suspicious transactions
* Model training, evaluation, and performance analysis
* Visualization of results using Python libraries
* Scalable approach for financial crime detection
